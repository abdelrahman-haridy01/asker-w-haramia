# عسكر المعاني

لعبة لغوية عربية مستوحاة من "عسكر وحرامية". يبحث اللاعب عن المعنى الصحيح للكلمة داخل الجملة اعتمادًا على السياق والصيغة.

## التشغيل

افتح `index.html` مباشرة في المتصفح.

## الرفع على GitHub Pages

المشروع ثابت ولا يحتاج إلى خطوة بناء. يمكن رفع الملفات التالية إلى الفرع المستخدم في GitHub Pages:

- `index.html`
- `styles.css`
- `app.js`

## المزايا الحالية

- شاشات البداية، المقدمة، التعليمات، اللاعبين، التحميل، اللعب، الفوز، الصدارة.
- قائمة داخلية وإعدادات صوت ومؤثرات وحجم نص وصعوبة.
- تحديات عربية مبنية على السياق والصيغة والدلالة.
- نظام نقاط وزمن وسلاسل إجابات صحيحة.
- حفظ نتائج الصدارة محليًا عبر `localStorage`.
- تصميم RTL متجاوب مع الجوال وسطح المكتب.
